JAVA语言的跨平台性

Java的良好跨平台性依赖于 Java 虚拟机：

Java的程序并不直接运行在操作系统上，而是运行在 Java 虚拟机上。而各操作系统都有自己版本的 Java 虚拟机，这也就意味着 Java 程序可以轻易在各种不同的操作系统上运行，也就实现了 Java 良好的跨平台性。

  • 注：JVM 本身不具备跨平台性，而是各种操作系统都有各自的 JVM 版本。

JRE 、 JDK 、 JVM

JRE

JRE ，即 Java Runtime Environment ，是 Java 程序运行时的环境，包括了 JVM 和运行时所需要的核心类库。

JDk

JDk ，即 Java Development Kit ， 是 Java 程序开发包，包含了 JRE 和开发工具。

JVM

就是Java虚拟机嘛。

方法

方法的定义

方法的使用类似与C语言中的函数。

方法使用的注意事项

  • 方法应该定义在类当中。
  • 方法中不能再定义方法，即方法的定义不能嵌套。
  • 方法若有返回值，必须写上 “return 返回值” 。

方法的重载

方法的重载，即overload。
特征：

  • 多个方法的名称一致，但是参数列表不同。

参数列表不同的具体描述：

  • 参数个数不同
  • 参数类型不同
  • 参数类型顺序不同
